# راهنمای استفاده از نمودار مقایسه‌ی ارز

این راهنما به شما کمک می‌کند تا **بدون پیش‌فرضی از ابزارهای فنی**، فایل `*.ipynb` را در VS Code اجرا کرده و نمودار تعاملی آن را مشاهده کنید.

---

## مرحله ۱: نصب VS Code

1. به سایت رسمی VS Code بروید: [https://code.visualstudio.com/](https://code.visualstudio.com/)  
2. روی دکمه **Download for Windows** کلیک کنید و فایل نصب را دانلود کنید.  
3. فایل دانلود شده را باز کنید و مراحل نصب را دنبال کنید:  
   - تیک گزینه **Add to PATH** را حتماً بزنید.  
   - بقیه گزینه‌ها را بدون تغییر رها کنید.  
4. پس از نصب، VS Code را اجرا کنید.

---

## مرحله ۲: نصب Python

1. به سایت Python بروید: [https://www.python.org/downloads/windows/](https://www.python.org/downloads/windows/)  
2. آخرین نسخه پایدار (Stable) را دانلود کنید.  
3. فایل نصب را اجرا کنید و **حتما گزینه "Add Python to PATH" را تیک بزنید**، سپس نصب را کامل کنید.

---

## مرحله ۳: نصب افزونه Jupyter در VS Code

1. در VS Code، روی آیکون **Extensions** (سمت چپ صفحه، شبیه مربع کوچک) کلیک کنید.  
2. در کادر جستجو، عبارت `Jupyter` را تایپ کنید.  
3. افزونه **Jupyter** (توسعه یافته توسط Microsoft) را پیدا و نصب کنید.  
4. همین‌طور می‌توانید افزونه **Python** را هم نصب کنید تا اجرای کد راحت‌تر شود.

---

## مرحله ۴: نصب pip و کتابخانه‌های مورد نیاز

برای اجرای فایل Notebook، ابتدا باید **pip** (ابزار نصب کتابخانه‌های پایتون) روی سیستم شما نصب باشد.

### ۴.۱: نصب pip در ویندوز

1. مطمئن شوید **Python نصب شده** و گزینه "Add Python to PATH" هنگام نصب تیک خورده است.  
2. مرورگر خود را باز کنید و فایل get-pip.py را دانلود کنید:  
   [https://bootstrap.pypa.io/get-pip.py](https://bootstrap.pypa.io/get-pip.py)  
3. فایل دانلود شده را در یک پوشه ساده (مثلاً `C:\PythonInstall`) قرار دهید.  
4. **Command Prompt** را باز کنید (کلید ویندوز + R → تایپ `cmd` → Enter).  
5. به مسیر فایل بروید، مثلا:

```bash
cd C:\PythonInstall
````

6. دستور زیر را اجرا کنید:

```bash
python get-pip.py
```

7. پس از پایان نصب، مطمئن شوید pip نصب شده است:

```bash
pip --version
```

---

### ۴.۲: نصب کتابخانه‌های مورد نیاز

#### با فایل `requirements.txt`:

* دستور زیر را اجرا کنید تا تمام کتابخانه‌ها به صورت خودکار نصب شوند:

```bash
pip install -r requirements.txt
```

---

## مرحله ۵: باز کردن فایل Notebook

1. در VS Code، از منوی **File > Open File**، فایل `*.ipynb` خود را انتخاب کنید.  
2. فایل باز می‌شود و هر سلول کد یک دکمه **▶️** برای اجرا دارد.  
3. روی دکمه **▶️** در بالای سلول‌ها کلیک کنید تا هر سلول اجرا شود.  

> ⚠️ ترتیب اجرای سلول‌ها مهم است: ابتدا سلول‌های بالاتر و سپس پایین‌تر اجرا شوند.

---

## مرحله ۶: مشاهده نمودار تعاملی

- پس از اجرای سلول مربوط به نمودار، نمودار تعاملی داخل VS Code نمایش داده می‌شود.  
- می‌توانید با ماوس روی نمودار حرکت کنید، زوم کنید و داده‌ها را بررسی کنید.  

---

## نکات مهم

- اگر نمودار یا داده نمایش داده نشد، دکمه‌ی Restart را زده و سلول‌ها را دوباره به ترتیب اجرا کنید.
